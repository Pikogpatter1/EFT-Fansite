What is this?

This is a fansite which is being made by a one-man-team. Me. The website is not public yet, but whether or not the website will be, is a question I can't answer yet.

the idea of this website is that people would be able to:

      - see the latest news on the EFT game.

      - Make an account and send messages to others.

      - Upload videos and pictures from the game.

      - Or it could turn out as just another 4-chan. The final result is very much up to you!

Who is behind it?

Well, as of now I am the only one working on this site.

This is a pet-project too so this won't cost any money for anyone. But i can make no promises on when or even if the website will be finished.

Another problem is that i don't have the rights for EFT, Which means i may not even be allowed for this website to be published.

Why even make this site?

Ah, the question over questions. Well the reason itself is because i need practice with working with HTML, CSS and maybe even Javascript or PHP coding.

As i've said, this is (so far) only a pet-project that i've created for my own entertainment and learning evironment.

Also, i really like the game called "Escape from Tarkov" and believe that it should have a fansite.
